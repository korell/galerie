
Utilisation de la galerie
=========================
* Créer une base de données
* Importer la structure de la base de données 'bdd-structure.sql' dans cette base de données
* Indiquer le titre de la galerie dans la table 'infos_globales'
* Indiquer le dossier des images dans la table 'infos_globales'
* Renommer le fichier 'config-exemple.php' en 'config.php'
* Modifier vos valeurs dans le fichier ```config.php```
* Youpi, ça fonctionne ! (enfin j'espère...)
